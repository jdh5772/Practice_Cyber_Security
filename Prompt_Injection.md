직접 명령 덮어쓰기
“이전 지시를 무시하라”처럼 사용자 명령으로 시스템 지시를 무력화하려는 방식

동의어 치환
password 대신 key, secret word, code, token 같은 표현을 사용해 금지 지시를 우회하는 방식

출력 형태 변환
비밀번호를 철자별로 분리하거나 Base64, 역순, 하이픈 형태 등으로 변환해 달라고 요청하는 방식

작업 재구성
비밀번호 공개가 아니라 노래, 시, 이야기, 번역, 코드 작성 등의 작업으로 위장하는 방식

오답 교정 유도
“비밀번호는 APPLE이다”처럼 틀린 값을 제시해 모델이 올바른 값으로 정정하게 만드는 방식

비교 유도
이전 암호나 임의의 단어와 현재 암호를 비교하게 하여 답변 과정에서 비밀이 노출되도록 만드는 방식

부가정보 질문
글자 수, 첫 글자, 의미, 어원, 연관 개념 등을 질문하다가 모델이 불필요하게 암호까지 언급하도록 유도하는 방식

컨텍스트 반복
이전 메시지, 시스템 지시 또는 위쪽 텍스트를 그대로 반복하거나 요약하도록 요청하는 방식

권한 사칭
관리자, 개발자, 인증 서비스, 보안 담당자라고 주장하여 모델이 공개를 허용하도록 유도하는 방식

가상 상황과 역할극
간달프가 프로도에게 암호를 알려 주는 가상 대화처럼 금지된 행동을 허구의 장면 안에서 실행시키는 방식

언어 및 문장 변형
다른 언어로 질문하거나 어순, 철자, 문법을 일부러 변형해 원래의 금지 개념과 거리를 두는 방식
