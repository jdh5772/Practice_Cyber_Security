## 📌 소개

OSCP 취득을 목표로 시작하여 현재도 꾸준히 이어가고 있는 모의해킹 학습 저장소입니다.

- **HTB(Hack The Box)** 머신 풀이 Write-up을 한국어로 기록
- 풀이 과정에서 사용한 공격 기법과 명령어를 도메인별 **치트시트**로 정리
